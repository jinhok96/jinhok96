# 강진호 (Kang Jinho)

비판적 사고와 긍정적 마인드를 갖춘 프론트엔드 개발자입니다.  
- 사용하고 이해하기 쉬운 UI/UX에 관심이 많습니다.
- 배우는 속도가 빨라 어떤 일이든 쉽게 습득하고 적응합니다.  
- 소통을 매우 중요하게 생각하며, 시각디자인을 전공하여 디자이너의 언어를 이해합니다.
- 문제 해결을 위해 토론과 의견 경청을 즐기며, 새로운 경험과 시도, 시행착오와 실패를 두려워하지 않습니다.
